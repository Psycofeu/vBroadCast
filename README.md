# vBroadCast message

__Pmmp custom broadcast message__

## Utils!
- [x] custom time
- [x] Custom Prefix
- [x] Custom messages (no limite)
- [x] Cutoms tags

## Tags:
|Tags|Description|
|:--:|:--:|
|`{line}`| Jump line |
|`{online}`| get online player |
|`{max_online}`| get Max online player |
